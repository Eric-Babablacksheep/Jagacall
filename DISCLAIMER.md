### 1. AI Accuracy Limitations

- **No 100% Accuracy**: AI analysis is not infallible and may produce false positives or false negatives
- **Context Dependent**: The AI may not understand full context, cultural nuances, or legitimate business practices
- **Evolving Scams**: Scammers constantly develop new tactics that may not be recognized by the AI
- **Language Limitations**: While optimized for English and Bahasa Malaysia, accuracy may vary with dialects, slang, or mixed languages

### 2. Technical Limitations

- **Internet Dependency**: Requires active internet connection for AI analysis
- **Processing Time**: Analysis may take several seconds depending on network conditions
- **File Size Limits**: File analysis is restricted to 10MB maximum
- **Audio Quality**: Voice analysis depends on transcript accuracy; poor audio quality affects results

### 3. Security Limitations

- **Not a Security Solution**: This app does not provide real-time call blocking or malware removal
- **No Biometric Authentication**: Voice analysis focuses on content patterns, not speaker verification
- **Data Privacy**: While we take privacy seriously, no digital system is completely immune to breaches

## üéØ Feature-Specific Disclaimers

### Call Detection

- **Transcript-Based**: Analysis is based on call transcripts, not live call monitoring
- **No Call Interception**: The app cannot intercept or block ongoing calls
- **Manual Input**: Users must manually provide call content for analysis
- **Verification Required**: Always verify suspicious calls through official channels

### File Analysis

- **Static Analysis**: Performs static file analysis; does not execute files for dynamic testing
- **Limited File Types**: Supports common file formats but may not detect all threat types
- **Not Antivirus**: Does not replace dedicated antivirus software
- **False Positives**: Legitimate files may be flagged as suspicious

### Voice Analysis

- **Transcript Dependent**: Relies on accurate speech-to-text conversion
- **No Voice Biometrics**: Does not analyze voice patterns for speaker identification
- **Content Focus**: Analyzes what is said, not how it is said
- **Cultural Context**: May not fully understand cultural-specific communication patterns

## üîí Best Practices

### For Phone Calls:

1. **Don't Rush**: Scammers create urgency to prevent rational thinking
2. **Verify Independently**: Call back using official phone numbers (not numbers provided by the caller)
3. **Never Share OTP**: One-Time Passwords are never requested by legitimate organizations
4. **Hang Up**: You have the right to end suspicious calls

### For Files:

1. **Scan First**: Use dedicated antivirus software before opening files
2. **Check Source**: Only download files from trusted sources
3. **Be Skeptical**: Unsolicited files are often malicious
4. **Update Software**: Keep your operating system and software updated

### For General Safety:

1. **Educate Family**: Teach elderly and young family members about scam tactics
2. **Stay Updated**: Follow official sources for the latest scam warnings
3. **Report**: Report scams to help protect others
4. **Trust Official Channels**: Government agencies and banks have official communication channels
